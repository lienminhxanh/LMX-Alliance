# Create New Business Sector

Create a complete business sector with:
1. Add new sector to Prisma seed file
2. Create slug from name: $ARGUMENTS
3. Generate placeholder content in VI/EN/ZH
4. Add to admin sectors list
5. Add route in public business-segments page

Sector name: $ARGUMENTS

Make sure:
- All 3 languages have content
- SEO metadata included
- Default status is DRAFT
- Order index is last
